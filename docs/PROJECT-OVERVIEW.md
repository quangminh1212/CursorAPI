# CursorPool API Adapter

Professional API adapter converting CursorPool to Claude API format.

## Structure

```
cursorpool-api/
├── src/              # Source code
├── examples/         # Usage examples  
├── scripts/          # Utility scripts
├── docs/             # Documentation
├── package.json      # Dependencies
├── README.md         # Main documentation
├── LICENSE           # MIT License
├── CHANGELOG.md      # Version history
└── CONTRIBUTING.md   # Contribution guide
```

## Quick Commands

```bash
npm start              # Start API adapter (port 8000)
npm test               # Run tests
npm run example        # Run examples
npm run show-tokens    # Display tokens
```

## Key Files

- `src/claude-api-adapter.js` - Main API adapter
- `examples/example-claude-api.js` - JavaScript example
- `examples/example-claude-api.py` - Python example
- `docs/CLAUDE-API-GUIDE.md` - Complete API guide
- `docs/QUICK-START.md` - 5-minute quickstart
